Overview

The compiler is implemented as a processing pipeline of primarily immutable data structures. Certain steps in the pipeline have dedicated class hierarchies (eg, ASTNode subclasses are for the parser), but much of the code attaches various “stuff” to existing structures (eg variable renaming sets a version field on Variable). Care has been taken to minimize the need for mutability, but sometimes it’s just impractical to do so.
